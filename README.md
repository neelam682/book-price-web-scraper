# Book Price Web Scraper

This Python script scrapes book data from a public website and exports it into a CSV file.

## Features
- Extracts book title
- Extracts price
- Extracts rating
- Saves results to CSV
- Uses requests, BeautifulSoup, and pandas

## Output
books.csv file containing structured data.

## Technologies Used
- Python
- requests
- BeautifulSoup
- pandas

## How to Run
Install dependencies:
pip install requests beautifulsoup4 pandas

Run script:
python scraper_books.py
